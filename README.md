# spring-example

Spring Framework のサンプルプログラムです。

## プロジェクト構成

```
spring-example-common   : 共通ライブラリ
spring-example-greeting : GET の検証サンプル
spring-example-person   : POST の検証サンプル
spring-example-app      : Spring Boot アプリケーション
```
